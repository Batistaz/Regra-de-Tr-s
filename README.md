Regra de Três Simples

Projeto desenvolvido com HTML, CSS e JavaScript para realizar cálculos de Regra de Três Simples de maneira rápida, prática e intuitiva.

🚀 Demonstração
A aplicação permite que o usuário informe três valores e descubra automaticamente o quarto valor proporcional.

🛠️ Tecnologias Utilizadas
HTML5 → estrutura da aplicação
CSS3 → estilização e layout
JavaScript → lógica do cálculo
📚 Conceito Matemático

A Regra de Três Simples é utilizada para encontrar um valor desconhecido em uma proporção.

✨ Funcionalidades
Entrada dinâmica de valores
Cálculo automático
Interface simples e responsiva
Resultado exibido em tempo real
Fácil de usar
